## abstract

- NROM
- mapper0
- joypad 1P
- Super Mario Bros works
- 60fps
- Joystick is OK (Xbox 360 Controller and Logitech Gamepad F310)
- Multi-platform (Win10 and Ubuntu18.04 is checked.)
- coredump is enabled

## usage

```golang
go run nes.go ./xxx.nes 
```
